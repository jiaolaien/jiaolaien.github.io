# [平行世界的我们](https://hoo.be/yingtaoyy)


咖啡馆里，林姝和对座的男士相谈甚欢。他是相亲对象，条件完美，无可挑剔。但她的思绪总飘向窗外，想起那个和她一起在深夜路边摊吃烧烤、笑得毫无形象的江辰。

她礼貌地结束约会，鬼使神差地走到了那个烧烤摊。江辰就在那里，坐在塑料凳上，朝她挥手，好像早知道她会来。

“怎么样？那个平行世界的我，是不是更符合你的预期？”他笑着问，递给她一串烤鸡翅。

林姝咬了一口，被辣得吸了口气，心里却无比踏实：“那个世界的他很好，但这个世界，有我的辣鸡翅。”

我们总会遇到看似“更合适”的人，但爱是，纵然知晓存在一万种幸福的可能，我依然坚定地选择这个有你的世界。

（选择：在所有的可能里，我只想要有你的这个现实）
